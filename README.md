# Space Data Analysis

## Project Overview
This repository contains solutions for astronomy and space data analysis problems using Python, Machine Learning, and Deep Learning.

---

# Problems Solved

## 1. Meteor Detection
Detect meteor events from antenna signal data using peak detection techniques.

### Techniques Used
- Signal Processing
- Peak Detection using scipy

---

## 2. Galaxy Redshift Prediction using CSV Data
Predict galaxy redshift values using photometric tabular data.

### Machine Learning Model
- Random Forest Regressor

### Features Used
- u
- g
- r
- i
- z

---

## 3. Galaxy Redshift Prediction using Images
Estimate galaxy redshift using galaxy image data.

### Deep Learning Model
- Convolutional Neural Network (CNN)

### Techniques Used
- Image Processing
- Deep Learning
- TensorFlow/Keras

---

## 4. Exoplanet Detection and Tidal Locking
Detect exoplanets using transit signals from stellar light curves and identify potentially tidally locked planets.

### Concepts Used
- Transit Method
- Light Curve Analysis
- Peak Detection
